# Sobre o Projeto : 

Este é o Vollmed-api, um projeto simples de cadastro de médicos, desenvolvido com um professor da *ALURA* com SpringBoot, JPA, Loombok, Flyway para migration com mysql.

Back end :

- Java
- SpringBoot
- JPA/Loombok
- Maven
- Flyway 

Banco de dados :
- MYSQL

Plataforma de API :
- Insomnia
